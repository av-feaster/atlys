# Atlys

A modern iOS application built with SwiftUI.

## Features

- **Infinite Carousel**: Smooth, scrolling carousel for onboarding or content display
- **Modern UI**: Built with SwiftUI for a clean, responsive interface
- **Onboarding Flow**: Intuitive user onboarding experience

## Requirements

- iOS 15.0+
- Xcode 13.0+
- Swift 5.5+

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/atlys.git
   ```
2. Open `atlys.xcodeproj` in Xcode
3. Build and run the project (⌘R)

## Project Structure

```
atlys/
├── atlys/                   # Main application target
│   ├── Assets.xcassets/     # App assets and icons
│   ├── feature/             # Feature modules
│   │   └── onboarding/      # Onboarding feature
│   ├── ContentView.swift    # Main app view
│   └── atlysApp.swift       # App entry point
└── atlys.xcodeproj/         # Xcode project files
```
